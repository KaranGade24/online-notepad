# Online Notepad

A simple web-based notepad that allows users to create, edit, and save text files with basic formatting options. It also shows a live word count and persists content in the browser using `localStorage`.

## Features

- **Create New File** – Clear the editor to start a new document.
- **Save File** – Download the content as a `.txt` file.
- **Basic Text Styling** – Bold and italic formatting using toolbar buttons.
- **Word Count** – Live word count displayed at the bottom.
- **Persistent Content** – Text remains even after page reload via `localStorage`.
- **File Menu** – Easy-to-access dropdown for file options.

## Technologies Used

- **HTML** – Structure of the editor.
- **CSS** – Styling and layout.
- **JavaScript** – Functionality for editing, formatting, saving, and counting words.
- **Font Awesome** – Icons for toolbar buttons.

## How to Use

1. Open [Live Preview](https://karangade24.github.io/online-notepad) OR `index.html` in your browser.
2. Start typing in the editor.
3. Use the toolbar buttons to make text **bold** or _italic_.
4. Use the **File** menu or the top toolbar to create a new file or save your text.
5. The word count updates automatically at the bottom.
